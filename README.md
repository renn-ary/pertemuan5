# Praktikum Bootstrap

Project ini merupakan tugas praktikum mata kuliah Web Development yang bertujuan untuk memahami dan mengimplementasikan framework Bootstrap dalam pembuatan website responsif.

---

## Deskripsi

Website ini dibuat menggunakan Bootstrap untuk mempelajari:

- Grid System
- Navbar
- Card Component
- Form Validation
- Responsive Layout
- Utility Classes

Project ini berfokus pada pembuatan tampilan website modern yang responsif di berbagai ukuran layar.

---

## Teknologi yang Digunakan

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Optional)
